HTML

Язык разметки для создания структуры веб-страниц

О языке

HTML (HyperText Markup Language) — это стандартный язык разметки, используемый для создания структуры веб-страниц. Он определяет элементы страницы, такие как заголовки, абзацы, ссылки и изображения.

Особенности

  • Простота в изучении и использовании
  • Совместимость со всеми браузерами
  • Поддержка мультимедиа (изображения, видео, аудио)
  • Интеграция с CSS и JavaScript

Пример кода


<!DOCTYPE html>
<html>
<head>
    <title>Пример HTML</title>
</head>
<body>
    <h1>Привет, Мир!</h1>
    <p>Это пример HTML-страницы.</p>
</body>
</html>
